บทที่ 1 — เริ่มต้นกับ Windows 10/11

ในบทนี้คุณจะได้เรียนรู้ตั้งแต่การติดตั้ง การตั้งค่าบัญชี และการตั้งค่าเริ่มต้นที่ควรรู้เพื่อทำให้เครื่องปลอดภัยและใช้งานได้อย่างเต็มประสิทธิภาพ

1. การติดตั้ง Windows (สรุปขั้นตอนพร้อมใช้งาน)

  1. ดาวน์โหลดไฟล์ติดตั้ง — ดาวน์โหลดเฉพาะจากเว็บไซต์ของ Microsoft เท่านั้น (เช่น Media Creation Tool สำหรับ Windows 10/11)
  2. เตรียม USB ติดตั้ง — ใช้ USB ที่ว่าง (อย่างน้อย 8GB) แล้วเปิด Media Creation Tool เพื่อสร้างตัวติดตั้ง
  3. บูตจาก USB — เสียบ USB แล้วรีสตาร์ทเครื่อง กดปุ่มที่กำหนด (F2, F12, ESC หรือ DEL ขึ้นกับยี่ห้อ) เพื่อเลือกบูตจาก USB
  4. ทำตามขั้นตอนการติดตั้ง — เลือกภาษา, ใส่ Product Key (หรือข้ามถ้ายังไม่มี), เลือกพาร์ทิชันและกดติดตั้ง
  5. หลังติดตั้ง — ระบบจะรีสตาร์ทอัตโนมัติ เข้า Windows แล้วทำการตั้งค่าพื้นฐานตามขั้นตอนด้านล่าง

2. การตั้งค่าบัญชีผู้ใช้ (Microsoft Account vs Local Account)

เมื่อเริ่มต้นใช้งานครั้งแรก Windows จะให้เลือกเข้าสู่ระบบด้วยบัญชี Microsoft หรือสร้าง Local Account

ข้อดีของบัญชี Microsoftข้อดีของ Local Account
ซิงก์การตั้งค่า, ใช้ OneDrive, เครือข่าย Microsoft Storeไม่ต้องเชื่อมต่อออนไลน์ เหมาะกับผู้ที่ต้องการความเป็นส่วนตัว
กู้คืนรหัสผ่านได้ง่ายผ่านอีเมล/เบอร์ไม่ต้องผูกบัญชีส่วนตัว

คำแนะนำ: ถ้าเป็นเครื่องส่วนตัวและต้องการความสะดวก ใช้ บัญชี Microsoft. หากต้องการความเป็นส่วนตัวหรือเครื่องที่ใช้ร่วม เลือก Local Account.

3. การตั้งค่าเริ่มต้นที่ควรรู้ (สิ่งที่ควรทำหลังติดตั้งทันที)

  • อัปเดต Windows: เปิด Settings → Windows Update แล้วคลิก Check for updates
  • สำรองข้อมูล: เปิด OneDrive หรือสำรองไฟล์สำคัญไปยัง External Drive
  • ตั้งค่าความปลอดภัย: เปิด Windows Security, Real-time protection และตั้งค่า Firewall
  • ติดตั้งซอฟต์แวร์จำเป็น: Browser, ZIP tool, Office, Media Player, Antivirus
  • ตั้งค่าแสดงนามสกุลไฟล์: File Explorer → View → Options → View → uncheck "Hide extensions"
  • ปรับ Startup apps: Task Manager → Startup → ปิดโปรแกรมไม่จำเป็น

4. วิธีอัปเดต Windows ให้ปลอดภัย (ฉบับเข้าใจง่าย)

  1. ไปที่ Settings → Windows Update แล้วคลิก Check for updates
  2. สำรองข้อมูลสำคัญก่อนอัปเดตใหญ่ (External Drive หรือ OneDrive)
  3. เปิดการอัปเดตอัตโนมัติที่ Advanced options
  4. อัปเดตไดรเวอร์จาก Windows Update หรือจากเว็บไซต์ผู้ผลิตเมื่อต้องการ
  5. หลีกเลี่ยงการดาวน์โหลดแพตช์จากแหล่งไม่น่าเชื่อถือ
  6. รีสตาร์ทเครื่องหลังอัปเดตเสมอ

5. เคล็ดลับสั้น ๆ สำหรับผู้เริ่มต้น

  • ใช้ Windows Restore Point ก่อนติดตั้งโปรแกรมสำคัญ
  • ตั้งรหัสผ่านหรือ PIN เพื่อรักษาความปลอดภัยบัญชีผู้ใช้
  • อย่าใช้บัญชี Administrator เป็นประจำ — สร้างบัญชี Standard สำหรับใช้งานปกติ
