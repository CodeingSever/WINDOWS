บทที่ 1 — เริ่มต้นกับ Windows 10/11

ในบทนี้คุณจะได้เรียนรู้ตั้งแต่การติดตั้ง การตั้งค่าบัญชี และการตั้งค่าเริ่มต้นที่ควรรู้เพื่อทำให้เครื่องปลอดภัยและใช้งานได้อย่างเต็มประสิทธิภาพ

1. การติดตั้ง Windows

  1. ดาวน์โหลดไฟล์ติดตั้งจากเว็บไซต์ Microsoft
  2. เตรียม USB ติดตั้ง (8GB+) ด้วย Media Creation Tool
  3. บูตจาก USB และทำตามขั้นตอนการติดตั้ง
  4. เลือกพาร์ทิชันและติดตั้ง
  5. หลังติดตั้งรีสตาร์ทและทำการตั้งค่าเริ่มต้น

2. การตั้งค่าบัญชีผู้ใช้

เลือกบัญชี Microsoft หรือ Local Account ตามความต้องการ

3. การตั้งค่าเริ่มต้นที่ควรรู้

  • อัปเดต Windows
  • สำรองข้อมูล
  • ตั้งค่าความปลอดภัย
  • ติดตั้งซอฟต์แวร์จำเป็น
  • แสดงนามสกุลไฟล์
  • ปรับ Startup apps

4. วิธีอัปเดต Windows Knowledge — เรียนรู้ Windows แบบพร้อมใช้งาน

1. การติดตั้ง Windows

  1. ดาวน์โหลดไฟล์ติดตั้งจากเว็บไซต์ Microsoft
  2. เตรียม USB ติดตั้ง (8GB+) ด้วย Media Creation Tool
  3. บูตจาก USB และทำตามขั้นตอนการติดตั้ง
  4. เลือกพาร์ทิชันและติดตั้ง
  5. หลังติดตั้งรีสตาร์ทและทำการตั้งค่าเริ่มต้น

2. การตั้งค่าบัญชีผู้ใช้

เลือกบัญชี Microsoft หรือ Local Account ตามความต้องการ

3. การตั้งค่าเริ่มต้นที่ควรรู้

  • อัปเดต Windows
  • สำรองข้อมูล
  • ตั้งค่าความปลอดภัย
  • ติดตั้งซอฟต์แวร์จำเป็น
  • แสดงนามสกุลไฟล์
  • ปรับ Startup apps
